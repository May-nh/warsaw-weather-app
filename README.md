# Warsaw Weather App

A short Java console application that shows the actual weather information in Warsaw at the moment of retrieval!

## Features!
- shows temperature, humidity, wind speed, and weather description
- live data from OpenWeatherMap
- console output

## Setup

1. Get a free API key from OpenWeatherMap (https://openweathermap.org/)
2. Set the API key as an environment variable
3. Build and run the project using Maven!
